\# Day 2 - Docker Installation



\## Steps

1\. Install Docker Desktop

2\. Start Docker



\## Verify Installation



docker --version

docker info



\## Check Containers



docker ps

docker ps -a



\## Service Commands



sudo systemctl start docker

sudo systemctl enable docker



\## Example



docker run hello-world



\## Notes

Docker requires virtualization support.



\## Summary

Docker must be properly installed before usage.
