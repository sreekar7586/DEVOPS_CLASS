\## Viva Questions and Answers



\### 1. What is Docker?

Docker is a containerization platform used to package applications with dependencies.



\### 2. Difference between Image and Container?

Image → static template  

Container → running instance  



\### 3. What is Dockerfile?

A file containing instructions to build Docker images.



\### 4. What is Volume?

Used to store persistent data outside container.



\### 5. What is Port Mapping?

Mapping host port to container port using -p option.



\### 6. What is Docker Compose?

Tool to manage multi-container applications.



\### 7. What is Microservices?

Architecture where application is divided into small services.



\### 8. What is CI/CD?

Automated build, test, and deployment pipeline.



\### 9. What is GitHub Actions?

Tool to automate workflows in GitHub.



\### 10. What is Docker Hub?

Repository to store Docker images.
